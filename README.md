# Designer Portfolio Website

A modern responsive personal portfolio website built using HTML, CSS, and JavaScript.
Designed for showcasing services, pricing plans, portfolio, and contact information in a clean professional layout.

## 🚀 Features

* Responsive Navbar with Hamburger Menu
* Hero Section with CTA Buttons
* About / Stats Section
* Services Showcase
* Pricing Plans
* Contact Form
* Social Media Footer
* Fully Mobile Responsive Design

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts
